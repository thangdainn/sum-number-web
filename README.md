📘 Dự án: Tính Tổng Hai Số

📌 Giới thiệu

Đây là một ứng dụng web đơn giản được xây dựng bằng Spring Boot và Thymeleaf, cho phép người dùng nhập vào hai số và tính tổng của chúng.

Ngoài ra, hệ thống còn hiển thị các bước tính toán chi tiết, giúp người dùng dễ hiểu cách ra kết quả.

🛠️ Công nghệ sử dụng
- Java 17+
- Spring Boot
- Thymeleaf
- HTML / CSS
- Maven

▶️ Cách chạy dự án
1. Clone project
````bash
git clone https://github.com/thangdainn/sum-number-web.git
cd sum-number-web
````
2. Chạy ứng dụng
````bash
mvn spring-boot:run
````
3. Mở trình duyệt
````bash
http://localhost:8080
````
